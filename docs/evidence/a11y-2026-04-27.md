## target-size (WCAG 2.5.5 AAA / 2.2 2.5.8 AA)

`e2e/target-size.spec.ts` で 3 件評価:

| テスト | 検証範囲 | しきい値 | 結果 |
|---|---|---|---|
| login 画面 | 全 visible button | WCAG AA (24×24 px) | **pass** |
| メイン画面 (login + 同意済) | 全 visible button | WCAG AA (24×24 px) | **pass** |
| 主要操作 button | 「開く」「SOAP を抽出する」「ログアウト」 | WCAG AAA (44×44 px) | **pass** |

`document.querySelectorAll('button')` を取得、`getBoundingClientRect()` で測定、`Math.min(width, height)` がしきい値以上であることを assert。Next.js dev overlay の button (`nextjs-portal` / `[data-nextjs-toast]`) は本番 build に出ない要素のため計測対象から除外している。
